# Calories Calculator

A simple and user-friendly web application that calculates the estimated daily calorie requirement based on a user's gender, age, height, and weight using the Mifflin-St Jeor Equation.

## Features

- Select Gender (Male/Female)
- Enter Age
- Enter Height (cm)
- Enter Weight (kg)
- Calculate estimated daily calorie needs
- Clean and responsive user interface
- Instant results using JavaScript

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html** or open it in your web browser.
4. Enter your details.
5. Click the **Calculate** button to view your estimated daily calorie requirement.

## Example

### Input

- Gender: Male
- Age: 20
- Height: 170 cm
- Weight: 65 kg

### Output

Daily Calories Needed: 1618 kcal

## Formula Used

### Male

Calories = (10 × Weight) + (6.25 × Height) − (5 × Age) + 5
```

Female

Calories = (10 x Weight) + (6.25 X Height) - (5 x Age) -161
```
